<h1 align="center">Documentação</h1>

A ideia da presente pasta é documentar decisões quanto à arquitetura do projeto, entre outras coisas.
Para representar de forma gráfica, farei uso do `Excalidraw`.


## Como visualizar arquivos `.excalidraw`:

1. Baixe o(s) arquivo(s) desejados no seu computador,
2. Acesse https://excalidraw.com/,
3. Vá no menú da interface e procure por `Open` ou insira o atalho `Ctrl + O`.
4. Selecione o arquivo que você baixou
5. Visualize.

> Se você usa a aplicação [Obsidian](https://obsidian.md/), instale o plugin do Excalidraw para uma visualização ainda mais fácil dos arquivos .excalidraw aqui presentes.
